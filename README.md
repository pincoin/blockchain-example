# PINCOIN

PINCOIN, toy blockchain and cryptocurrency made in Python/Django 

# Requirements

* Django
* djangorestframework
* channels
* channels_redis
* websockets
* django-model-utils==3.2.0

# Cryptocurrency
* Transactions (Confirmations)
* Proof of Work (Mining)
* Wallets
* Decentralization
* P2P with WebSockets
* Blockchain Explorer
* REST API backends
